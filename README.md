# pdf-report-gen

Gerador de relatórios do banco de dados em formato de PDF a fim de estudos com NodeJS.

Módulos utilizados:
- Prisma
- Express
- PDFMake
- Moment

Linguagem:
- Typescript

Banco de dados:
- Postgres
